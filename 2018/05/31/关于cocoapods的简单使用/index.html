一.cocoapods简介

Cocoapods是OS X 和iOS下的一个第三方类库的管理工具，通过它我们可以为我们的项目添加‘pods’依赖库，轻松的管理三方类库的版本。并且在引入第三方类库时他可以为我们帮助我们完成基本的配置，包括编译阶段、连接器选择，甚至是ARC环境下的-fno-objc-arc配置等等。我们也可以使用cocoapods查找最新的三方库的版本，只需要使用pod search “三方库名称”即可。

二.安装步骤

1.查看当前的Ruby环境，并且升级它

发现还是有错误的

但是这个地方有个是我的gem已经升级成功了

2.我直接开始执行下面的替换源（暂时跳过了上面的错误的步骤）

之前使用的淘宝的源现在也不推荐使用了，所以这里使用的是ruby-china提供的镜像服务，执行命令
gem sources –add https://gems.ruby-china.org/ –remove https://rubygems.org/

替换源成功，然后查看原路径是否替换成功
执行命令 gem sources -l

接下来开始安装Cocoapods 执行命令
sudo gem install cocoapods 然后不出意外 又报错了

如果出错的话，我们就需要执行另外一个终端命令行
sudo gem install -n /usr/local/bin cocoapods

安装成功之后就要执行最为耗时的一步了
执行命令 pod setup 这个命令是设置仓库，把所有的上传到cocoapods的第三方框架下载每一个版本和网络地址以及其他信息到本地。

这个过程有点长 提示到setup completed时 说明已经完成了
