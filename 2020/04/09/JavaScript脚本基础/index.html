JS 简介

JavaScript是一种解释型的、基于对象的脚本语言，它引进了Java语言的概念，是内嵌于HTML中的脚本语言。

JavaScript具有以下的语言特点：

  • JavaScript是一种脚本编写语言，采用小程序段的方式实现编程，开发过程非常简单。
  • JavaScript是一种基于对象的语言，它能运用已经创建的对象。
  • JavaScript是一种基于Java基本语句和控制流之上的简单而紧凑的设计语言，比较简单，其次它的变量类型采用弱类型，并未使用严格的数据类型。
  • JavaScript是动态的，可以直接对用户的输入做出响应，无需经过Web服务程序。
  • JavaScript是一种安全性语言，它不允许访问本地磁盘，并且不能将数据存入服务器，不允许对网络文档进行修改和删除，只能通过浏览器实现信息浏览或动态交互，从而有效的防止数据丢失。
  • JavaScript具有跨平台性，它能依赖于浏览器本身，与操作环境无关。

JavaScript语言也有自己的常量、变量、表达式、运算符、以及程序的基本框架。其中常量中的特殊字符是以反斜杠/开头的不可系那是的特殊字符。其他的类型大部分同其他的基本编程语言类似，也有自己的条件控制语句等。

JavaScript的事件

JavaScript是基于对象的语言，而基于对象的基本特征，就是采用事件驱动。其中常用的事件有：

  • onClick事件 鼠标单击事件，是最常用的事件之一。
  • onChange事件 一个与表单相关的事件，当利用text或textarea元素输入的字符值改变时发生该事件，同时select表格中的一个选项状态发生改变后也会引发该事件。
  • onSelect事件 当文本框中的内容被选中时所发生的的该事件
  • onFocus事件 当单击表单对象时，即将光标放在文本框或选择框上时产生onFocus事件。
  • onLoad事件 当加载网页时，会产生该事件，它的作用是在首次载入一个页面文件时检测cookie的值，并用一个变量为其赋值，使其可以被源代码使用
  • onBlur事件 失去焦点事件，正好和获得焦点事件相对，当text、textarea对象或者select对象不再拥有焦点退到后台时，引发该事件
  • onMouseOver事件 当鼠标指针移动到某对象范围的上方时触发的事件
  • onMouseOut事件 当鼠标指针离开某对象范围时触发的事件
  • onDblClick事件 当鼠标双击时触发该事件

其他的也有许多常用的事件，这里就不一一列举了。下面我们介绍一下浏览器的内部对象以及常用属性。

浏览器的内部对象

使用浏览器的内部对象，可以实现与HTML文档的交互。浏览器的内部对象主要包括以下几个。

  • 浏览器对象（navigator）：提供有关浏览器的信息
常用属性 说明
appName 浏览器的名称
appVersion 浏览器的版本
appCodeName 浏览器的代码名称
browserLanguage 浏览器所使用的语言
plugins 可以使用的插件信息
platform 浏览器系统所使用的平台，如win32等
cookieEnabled 浏览器的cookie功能是否打开
  • 文档对象（document）： document对象包含了与文档元素一起工作的对象
对象 说明
anchor锚对象 是指在a name= 标记在HTML源代码中存在时产生的对象，它包含着文档中所有的anchor信息
links链接对象 是指在a href= 标记链接一个超文本或超媒体的元素作为一个特定的URL
form窗体对象 是文档对象的一个元素，它含有多种格式的对象储存信息，使用它可以在JavaScript脚本中编写程序，并可以用来动态改变文档的行为
write（）、writeln（） 主要用来实现在Web页面上显示输出信息
  • 窗口对象（windows）：windows对象处于对象层次的最顶端，它提供了处理浏览器窗口的方法和属性

windows对象处于对象层次的最顶端，它提供了处理navigator窗口的方法和属性。

方法 方法的含义和参数说明
open(url,windowName,parameterlist) 创建一个新的窗口，3个参数分别用于设置URL地址，窗口名称和窗口打开属性（一般包括宽度、高度、定位、工具栏等）
close（） 关闭一个窗口
alert（text） 弹出式窗口，text参数为窗口中显示的文字
confirm（text） 弹出确认域 text参数为窗口中的文字
promt（text，defaulttext） 弹出提示框 text为窗口中的文字，defaulttext参数用来设置默认情况下显示的文字
moveBy（水平位移，垂直位移） 将窗口移至指定的位移
moveTo（x，y） 将窗口移动到指定的坐标
resizeBy（水平位移，垂直位移） 按给定的位移量重新设置窗口大小
resizeTo(x，y) 将窗口设定为指定大小
back（） 页面后退
forward（） 页面前进
home（） 返回主页
stop（） 停止装载网页
print（） 打印网页
status 状态栏信息
location 当前窗口URL信息
  • 位置对象（location）：location对象提供了与当前打开的URL一起工作的方法和属性，它是一个静态的对象
属性 实现的功能
protocol 返回地址的协议，取值为http：、https、file：等
hostname 返回地址的主机名，例如http://www.baidu.com/china地址的主机名为www.baidu.com
port 返回地址的端口号，一般http的端口号是80
host 返回主机名和端口号，如www.baidu.com：8080
pathname 返回路径名，如http://www.baidu.com/d/index.html的路径为d/index.html
hash 返回”#“以及其后的内容，如果地址里面没有”#“，则返回空字符串
search 返回”？“以及其后的内容，如果地址里面没有”？“，则返回空字符串
href 返回整个地址，即返回在浏览器的地址栏中显示的内容
  • 历史对象（history）：history对象提供了与历史清单有关的信息
常用方法 说明
back() 后退，与单击”后退“按钮是等效的
forward() 前进，与单击”前进“按钮是等效的
go() 该方法是用来进入指定的页面
