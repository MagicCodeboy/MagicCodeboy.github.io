#表格

表格

表格是网页制作中使用的最多的工具之一。在制作网页时，使用表格可以更清晰的排列数据，但在实际制作过程中，表格更多的用于网页的布局定位。很多网页都是以表格布局的，这是因为表格在文本和图像的位置控制方面都有很强的功能。

表格由行、列和单元格三部分组成，一般通过3个标记来创建，分别是表格标记table、行标记br和单元格标记td。表格的各种属性都要放在table元素之间。

创建表格

表格有许多的属性来供我们选择，合理的使用表格的各种属性可以更好的设计我们想要的表格效果。

caption表格标题

我们可以使用caption来设置标题单元格，表格的标题单元格一般位于整个表格的第1行，一个table表格只能含有一个表格标题。

表头th

表格的表头th是td单元格的一种变体，实质上仍然是一种单元格。它一般位于第一行和第一列，用来标明这一行或列的内容类别，一般情况下，浏览器会以粗体和居中的样式显示th元素中的内容。

表格的基本属性

同列表一样，表格也有自己的基本属性，我们可以通过设置这些基本属性来达到我们想要的效果。我们也可以使用相关属性来设置行列的样式。

属性 属性值
width 表格宽度
height 表格高度
align 表格的对齐方式值一般有三种，left、right、center.这个是设置整个表格在浏览器中的对齐方式，如果要设置行、列的对齐方式，需要在行或者列单元格内另外定义。在tr标记中用用来控制当前行的水平对齐方式
border 表格的边框，只有设置border的值不为0，才能在网页中显示出表格的边框
bordercolor 表格的边框颜色
cellspacing 内框宽度，用于设置表格内部每个单元格之间的间距
cellpadding 表格内文字与边框间距，设置单元格边框与单元格里的内容之间的间距
bgcolor 表格的背景颜色
background 表格的背景图像
valign 在tr标记中用来控制表格当前行的垂直对齐方式，属性值有三种，分别是top、middle、bottom.

单元格属性

单元格的宽高属性设置也是通过width和height属性来设置的，它也有特别的属性来设置单元格的属性。

属性 属性值
colspan 水平跨度，可以设置两个或者多个相邻的单元格组合成一个单元格。
rowspan 垂直跨度，与上面的水平跨度对应，可以设置单元格在垂直方向上跨行的个数。
align 对齐方式，属性值left、right、center
valign 单元格的垂直对齐方式，属性值是top、middle、bottom
bordercolorlight 单元格的亮边框，就是单元格边框的向光的部分，可以设置单元格亮边框的颜色
bordercolordark 单元格的暗边框，就是单元格边框背光的部分，可以设置单元格暗边框的颜色

表格的结构

也有一些标记是用来表示表格的结构的，包括表首标记thead、表主体标记tboty和表尾标记tfoot,用于整体规划表格的行列属性，使用这些标记可以对表格的一行或者多行单元格属性进行统一的修改，从而省去了逐一修改单元格属性的麻烦。

表单

表单主要用来收集客户端提供的相关信息，使网页具有交互功能。在制作网页的过程中，常常需要使用表单，如会员注册、网上调查和搜索等。访问者可以使用如文本域、列表框、复选框以及单选按钮之类的对表单对象输入信息，然后单击某个按钮提交这些信息。表单的标记是form，在form标记中可以设置表单的基本属性，包括表单的名称、处理程序和传送方法等。一般情况下，表单的处理程序action和传送方法method是必不可少的参数。表单的处理程序是表单要提交的地址，也就是表单中收集到的资料要传递的程序地址。这一地址是绝对地址，也可以是相对地址，还可以是一些其他形式的地址。

表单的属性：

属性 属性值
name 给表单命名，不是表单的必要属性，为了防止表单提交到后台处理程序时出现混乱，一般需要给表单命名
method 指定数据提交到服务器的时候使用哪种HTTP提交方法，可取值get、post。get：表单数据被传送到action属性指定的URL，然后这个新的URL被送到处理程序上。post：表单数据被包含在列表主体中，然后被送到处理程序上
enctype 设置表单信息提交的编码方式。可取值分别是：application/x-www-form-urlencoded:默认的编码方式 multipart/form-data:MIME编码，上传文件的表单必须选择此项
target 用来指定目标窗口的打开方式，表单的目标窗口往往用来显示表单的返回信息。取值分别为_blank、_parent、_self和_top
