CSS简介以及常用样式介绍

CSS简介

CSS(层叠样式表)是一种网页制作技术，是网页设计比不可少的工具之一。网页最初是使用HTML标记来定义文档及格式，但这些标记不能满足更多的文档样式需求。为了解决这个问题，在1997年W3C在颁布HTML4的同时，也公布了有关样式表的第一个标准CSS1，在1998年又发布了CSS2版本，样式表得到了更多的充实。使用CSS能够简化网页的格式代码，加快加载显示的速度，也减少了需要上传的代码数量。它可以把网页上的内容结构和格式控制相分离。CSS可以更加精确地控制网页的内容形式，CSS的样式也是丰富多彩的，CSS的定义样式灵活多样，可以根据不同的情况，选用不同的定义方法。

CSS的基本语法

CSS的语法结构由三部分组成：选择符、样式属性和值。
选择符{样式属性：值；样式属性：值；…},添加CSS的方法有四种，分别是：链接外部样式表、内部样式表、导入外部样式表和内嵌样式。

链接外部样式表方法：link rel=stylesheet type=text/css href=style.css

内部样式表：内部样式表一般位于HTML文件的头部，在head标签内部 并且包含在style标签中

导入外部样式表：导入外部样式表是指在内部样式表style里导入一个外部样式表，导入时使用@import，实质上它是相当于存在于内部样式表中。

字体的属性

属性 属性值
font-family 字体 取值：如宋体，可以设置多个字体属性，如果不支持设置的第一个字体，则会往后查找可用的字体属性
font-size 字号 取值：xx-small 绝对字体尺寸 最小， x-small 绝对字体尺寸 较小，small 绝对字体尺寸 小，medium 绝对字体尺寸 正常默认值，large 绝对字体尺寸 大，x-large 绝对字体尺寸 较大，xx-large 绝对字体尺寸 最大 ，larger 相对字体尺寸 相对于父对象中字体尺寸进行相对增大，smaller 相对字体尺寸 相对于父对象中字体尺寸进行相对减小， length 可采用百分比或长度值，不可为负值，其百分比取值是基于父对象中的字体的尺寸。
font-style 字体风格 取值：normal 正常字体； italic 斜体显示文字；oblique 属于中间状态，以偏斜体显示。
font-weight 加粗字体 取值：normal 正常粗细；bold 表示粗体；bolder 表示特粗体；lighter 表示特细体；number 不是真正的取值，其范围是100~900，一般情况下都是整百的数字
font-variant 小写字母转大写 可以将小写的英文字母转化为大写。 取值：normal 正常显示；small-caps 将小写字母转化为大写字母并且字体较小。
复合属性font 设置font的复合属性， 各个值之间用空格相连。

颜色和背景属性

属性 属性值
color 用来指定元素的颜色，颜色值是一个关键字或者是一个十六进制的RGB值
background-color 不但可以设置网页的背景颜色，还可以设置字体的背景颜色
background-image 设置元素的背景图像 图像可以使绝对地址，也可以是相对地址
background-repeat 设置背景图像是否平铺，并且可以设置如何平铺。取值：no-repeat 背景图像不平铺； repeat 背景图像平铺排满整个网页；repeat-x 背景图像只在水平方向上平铺；repeat-y 背景图像只在垂直方向上平铺
background-attachment 可以设置背景图像是随对象滚动还是固定不动。取值：scroll 表示背景图像随对象滚动而滚动，是默认选项；fixed 表示背景图像固定在页面上不动，只有其他的内容随滚动条滚动
background-position 设置背景图像的位置 只能应用于块级元素和替换元素 替换元素包括img、input、textarea、select和object 可以组合使用top left right bottom来进行位置设置或者使用百分比来进行设置 或者设置x 和 y值
复合属性background 设置background的复合属性， 各个值之间用空格相连。

段落属性

属性 属性值
word-spacing 单词间隔 可以使用normal也可以使用长度值 取值：normal 正常的间隔 是默认选项；长度是设置单词间隔的数值及单位，可以使用负值
letter-spacing 字符间隔 设置属性可以设置数值及单位
text-decoration 文字修饰 可以对文本进行修饰，如设置下划线、删除线等 取值：normal 表示不修饰 是默认值； underline 表示对文字添加下划线；overline 表示对文字添加上划线；linethrough 表示对文字添加删除线；blink 表示文字闪烁效果
vertical-align 垂直对齐方式 取值：baseline 浏览器默认的垂直对齐方式；sub 文字的下标；super 文字的上标；top 垂直靠上对齐；text-top 使元素和上级元素的字体向上对齐；middle 垂直居中对齐；text-bottom 使元素和上级元素的字体向下对齐
text-transform 文本转换属性 用来转换英文字母的大小写。取值：none 表示使用原始值；capitalize 表示每一个单词的第一个字母大写；uppercase 表示使每个单词的所有字母大写；lowercase 表示使每个单词的所有字母小写
text-align 水平对齐方式 取值left 左对齐；right 右对齐；center 居中对齐；justify 两端对齐
text-indent 文本缩进 在HTML中只能控制段落的整体向右缩进，如果不进行设置，则浏览器默认为不缩进，在CSS中可以控制段落的首行缩进以及缩进的距离。缩进值必须是一个长度值或者一个百分比。
line-height 控制段落中行与行之间的距离 行高值可以设置为长度、倍数或者百分比。
white-space 处理空白 用于设置页面内空白的处理方式。取值：normal 默认属性 将连续的多个空格合并；pre 会导致源代码中的空格和换行符被保留，单只有在IE6中才能正确显示；nowrap 强制在同一行内显示所有文本，直到文本结束或者遇到br标签
unicode-bidi 文本反排 与下面的direction属性经常一起使用 用来设置对象的阅读顺序。 取值：bidi-override 表示严格按照direction属性的值重排序；normal 表示为默认值；embed 表示对象打开附加的嵌入层，direction属性的值指定嵌入层，在对象内部进行隐式重排
direction 与上面的unicode-bidi经常一起使用，来设置文本反排。 取值： ltr表示从左到右的顺序阅读；rtl表示从右到左顺序阅读；inherit 表示文本流的值不可继承

外边距、内边距以及其他边距

使用外边距可以设置元素周围的边界宽度，包括上下左右四个边界的距离；内边距属性用于设置边框和元素内容之间的间距，同样包括上下左右四个方向的边距值。

属性 属性值
padding 内边距 属性值包括padding-top、padding-bottom、padding-left、padding-right
margin 外边距 包括margin-top、margin-bottom、margin-left、margin-right

边框属性border
border|边框 属性值有border-style、border-width和border-color

属性 属性值
border-style 取值：none 默认值，无边框；dotted 点线边框；dashed 虚线边框；solid 实现边框；double 双实线边框；groove 边框具有立体感的沟槽；ridge 边框成脊形；inset 使整个边框凹陷，即在边框内嵌入一个立体边框；outset 使整个边框凸起 即在边框外嵌入一个立体边框
border-width 可以使用长度来设置边框的宽度 也可以使用属性有：medium 表示默认宽度；thin 表示小于默认宽度；thick 表示大于默认宽度
border-color 可以使用16中颜色或者RGB值来设置 也可以单独设置上下左右四个边框的颜色，如border-right-color：red

定位属性

使用定位属性可以控制元素的位置，包括相对定位和绝对定位两种方式。相对定位是指允许元素在相对于文档布局的原始位置上进行偏移，而绝对定位是指允许元素与原始的文档布局分离且任意定位。

属性 属性值
position 定位方式 取值：static 默认值；absolute 表示采用绝对定位，需要同时使用left、right、top、bottom等属性进行绝对定位；fixed 表示当页面滚动时，元素不随着滚动；relative 表示采用相对定位，对象不可层叠
top、left、right、bottom 元素位置 auto表示默认值；长度值包含数字和单位，也可以使用百分比来进行设置
z-index 层叠顺序 使用层叠顺序可以设置层的先后顺序和覆盖关系。默认情况下，z-index的值为1，表示该层位于最底层。
float 浮动属性 可以设置文字在某个元素的周围，它能应用于所有的元素 取值：none 表示对象浮动；left 表示文字浮动在元素的右边；right 表示文字浮动在元素的左边
clear 清除属性 用于指定一个元素是否允许有其他元素漂浮在它的周围。取值：none表示允许两边都可以有浮动对象，是默认值；left 表示不允许左边有浮动对象；right 表示不允许右边有浮动对象；both 表示不允许有浮动对象
clip 可视区域 可以吸纳定只显示裁切出的区域，裁切出的区域为矩形，只要设置两个点即可，一个是矩形左上角的顶点，有top和left两项设置完成，另一个是右下角的顶点，由bottom和right两项设置完成。 auto表示对象不裁切，是默认值。rect（数字）表示它依据上、右、下、左的顺序提供自对象左上角坐标计算的四个偏移数值，其中任一数值都可以用auto替换，即此边不裁切。
width、height 层的宽度和高度
overflow 超出范围 可以设置当层内的内容超出层所能容纳的范围时的显示方式，取值：visible 表示层的大小和内容都会自动显示出来；auto 表示只在内容超出层的范围时才会显示滚动条；hidden 表示会隐藏超出层范围的内容；scroll 表示总是显示滚动条
visibility 可见属性 是针对嵌套层的设置，嵌套层是插入在其它层中的层，分为嵌套的层和被嵌套的层，取值：inherit 表示继承父对象的可见性，是默认值；visible 表示无论父层对象是否可见，子层都是可见的；hidden 表示无论父层对象是否可见，子层都隐藏

列表属性

列表属性可以设置列表项的样式，包括符号、缩进等。

属性 属性值
list-sytle-type 取值：disc 默认值 实心圆；circle 空心圆；square 实心方块；decimal 阿拉伯数字；lower-roman 小写罗马数字；upper-roman 大写罗马数字；lower-alpha 小写英文字母；upper-alpha 大写英文字母；none 不适用任何项目符号或者编号。
list-style-image 图像符号属性 是指使用图像作为列表符号，以美化网页。none表示不指定图像，设置url属性可以使用绝对地址或者相对地址来指定符号的图像
list-style-position 列表缩进属性可以设置列表缩进的程度。 取值：outside 表示列表项目标记放置在文本以外，且环绕文本不根据标记对齐；inside 表示列表项目放置在文本以内，且环绕文本根据标记对齐

光标属性

使用光标属性可以设置在对象上移动时鼠标指针采用的光标形状。

取值 含义
default 客户端平台的默认光标。通常是一个箭头
hand 竖起一只手指的手型光标
crosshair 简单的十字形光标
text 大写字母I的形状
help 带有问号标记的箭头
wait 用于标示程序忙用户需要等待的光标
e-resize 向东的箭头
ne-resize 向东北的箭头
n-resize 向北的箭头
nw-resize 向西北的箭头
w-resize 向西的箭头
sw-resize 向西南的箭头
s-resize 向南的箭头
se-resize 向东南的箭头
auto 默认值。浏览器根据当前情况自动确定光标类型

滤镜的属性

CSS 中的滤镜和Photoshop中的滤镜相似，它可以用很简单的方法对网页中的对象进行特效处理，使用滤镜属性可以把一些特殊效果添加到网页元素总，使页面更加美观。但是，filter滤镜是比较早的CSS1属性，IE10起已经不再支持。

属性 属性值
alpha 不透明度 取值：opacity 开始时的透明度，取值范围为0~100，默认值为0，即完全透明，100为完全不透明；finishopacity 结束时的透明度 取值范围为0~100；style 设置渐变的样式，其中0表示无渐变，1为直线渐变，2为圆形渐变，3为矩形渐变；startx 设置透明渐变开始点的水平坐标。其数值作为对象宽度的百分比处理，默认值为0；starty 设置透明渐变开始点的垂直坐标；finishx 设置透明渐变结束点的水平坐标；finishy 设置透明渐变结束点的垂直坐标
blur 用于设置对象的动态模糊效果 取值：add 设置是否显示原始图片；direction 设置动态模糊的方向，按顺时针的方向以45度为单位进行累积；strength 设置动态模糊的强度 只能使用整数来指定，默认是5个
chroma 对颜色进行透明处理 是将图片中的某种颜色转换为透明色，产生透明效果
dropShadow 用于设置在指定的方向和位置上产生阴影效果 color控制阴影的颜色 offX和offY分别设置阴影相对于原始图像移动的水平距离和垂直距离；positive属性设置阴影是否透明
flipH、flipV 用于设置沿水平方向翻转对象和沿垂直方向翻转对象
glow 用于设置在对象周围产生发光的效果 color用于设置发光的颜色；strength用于设置发光的强度，取值范围为1~255，默认值为5
gray 灰度处理 用于把彩色图片中的色彩去掉，转换为黑白图片
invert 反相 用于设置对象反相，可以将图片的颜色、饱和度和亮度完全反转过来
xray X光片效果 用于制作类似X光片的效果
mark 用于为对象产生遮罩效果，可以做出像印章一样的效果 color可以设置外围遮盖的颜色
wave 波形滤镜 用于为对象内容创建波纹扭曲效果 属性值设置add用于设置是否要把对象按照波形样式打乱；freq 用于设置图片上的波浪数目；lightstrength 用于设置波浪的光照强度，取值范围为0~100；phase 用于设置波浪的起始位置；strength用于设置波浪的强度大小。
